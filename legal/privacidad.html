Política de privacidad

Información sobre el tratamiento de datos personales en Gadgetsmania.

Responsable del tratamiento

El responsable de esta web es Gadgetsmania.org. Para cualquier consulta relacionada con los datos personales, puedes contactar a través de la página de contacto.

Datos que se recogen

En esta web no se recogen datos personales directamente, salvo que el usuario contacte de forma voluntaria mediante correo electrónico.

Finalidad

Los datos facilitados se utilizarán exclusivamente para responder a consultas o comunicaciones enviadas por el usuario.

Cookies

Esta web puede utilizar cookies para mejorar la experiencia del usuario y analizar el tráfico. Puedes consultar más información en la política de cookies.

Publicidad y afiliación

Esta web participa en programas de afiliados (como Amazon) y puede mostrar publicidad. Esto significa que algunos enlaces pueden generar una comisión sin coste adicional para el usuario.

Derechos del usuario

El usuario puede solicitar acceso, rectificación o eliminación de sus datos en cualquier momento contactando a través de los medios disponibles en la web.

Actualizaciones

Esta política puede actualizarse en cualquier momento para adaptarse a cambios legales o técnicos.
